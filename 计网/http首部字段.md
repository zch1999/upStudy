# http 首部字段

- 通用首部

  1. cache-control 缓存控制
  2. connection 连接管理
  3. transfor-Encoding 报文主体的传输编码格式

- 请求首部

  1. Accept 客户端或者代理能够处理的媒体类型
  2. if-None-Match 比较实体 Etag
  3. if-Modified-Since 比较资源的更新时间
  4. Authorization 认证信息

- 响应首部

  1. Accept-Ranges 能接受的字节范围
  2. loacation 重定向的 URI
  3. Etag 资源唯一标识符
